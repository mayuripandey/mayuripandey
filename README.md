# 💫 About Me:
I’m Mayuri Pandey, a Full-Stack AI Engineer and Data Scientist based in Germany. I build production-oriented solutions across Generative AI, machine learning, LLM applications, RAG systems, forecasting, recommendation systems, and intelligent document processing.<br><br>My work combines Python, FastAPI, LangChain, LangGraph, PostgreSQL, Docker, and cloud technologies to develop reliable, scalable, and business-focused AI products.<br><br>I’m particularly interested in agentic AI, advanced retrieval architectures, LLM evaluation, MLOps, and applied AI systems that solve real operational problems.<br>
